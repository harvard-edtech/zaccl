Add one alt-host if not already in the list. If another user in the alt-host list has been deactivated, all alt-hosts are removed and the requested user is added as the only alt-host
Parameters:
Name Type Description
options object object containing all arguments
Properties
Name Type Attributes Description

altHost string the email or id of the alt host to be added
meetingObj object <optional>
 Zoom meeting object that needs to be updated. Meeting ID is not used to fetch meeting if this object is passed
Returns:
updated meeting object after adding alternative host
